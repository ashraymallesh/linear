# linear-regression
implementing a linear regression model from scratch in python (without using scikit-learn)

sv-linreg.py is the single variable version
mv-linreg.py is the multivariable version (for multiple features)

# Requirements
Numpy `pip3 install numpy` on MacOS/Linux
Numpy `python -m pip install numpy` on Windows

Matplotlib `pip3 install matplotlib` on MacOS/Linux
Matplotlib `python -m pip install matplotlib` on Windows
